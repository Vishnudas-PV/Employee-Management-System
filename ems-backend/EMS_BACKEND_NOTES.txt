           EMS BACKEND
       -------------------


  1.Create a new database in mongodb compass and create a new collection named as emnployees
  2.Create a new folder named as ems-backend
  3.Create a pacjage.json file -  npm init -y
  4.Install express,cors,mongoose - npm i express cors mongoose          
  5.Create index.js
  6.Create a new folder named as services
  7.Create a db.json file inside the services folder
  8.Create a new folder named as logic.js inside the services folder(for implementing all the logics)
